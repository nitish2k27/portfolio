# Nitish Patil - Portfolio

A modern, responsive portfolio website built with a neo-brutalist design aesthetic.

##  Features

- **Responsive Design**: Works perfectly on all devices (mobile, tablet, desktop)
- **Neo-Brutalist Aesthetic**: Bold typography, vibrant colors, and strong visual hierarchy
- **Interactive Sections**: Home, About, Resume with smooth transitions
- **Project Showcase**: Horizontal scrolling project gallery
- **Skills Animation**: Dynamic skill boxes with GSAP animations
- **PDF Resume Viewer**: Embedded resume with expand/collapse functionality
- **Contact Integration**: Direct links to LinkedIn, GitHub, WhatsApp, and email

##  Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with Tailwind CSS
- **JavaScript** - Interactive functionality
- **GSAP** - Smooth animations and scroll triggers
- **Responsive Design** - Mobile-first approach

##  Responsive Compatibility

-  Mobile (320px - 768px)
-  Tablet (768px - 1024px) 
-  Desktop (1024px+)
-  Cross-browser compatible

##  Design Highlights

- **Color Scheme**: Coral-red (#ee6055) with black grid patterns
- **Typography**: Inter font with Caveat handwritten accents
- **Layout**: Grid-based responsive design
- **Animations**: Smooth GSAP-powered transitions
